### 1단계 - 기능 구현 목록

- [x] 등록된 음식점 리스트를 확인할 수 있다.
- [x] 음식점 리스트를 정렬 가능하다.
  - [x] 거리순
  - [x] 이름순
- [x] 음식점을 카테고리로 필터링 할 수 있다.
  - [x] 새로운 음식점을 추가할 수 있다.
    - [x] 카테고리, 이름, 거리는 입력 필수
    - [x] 카테고리는 "한식", "중식", "일식", "아시안", "양식", "기타" 중 하나를 선택
  - [x] 거리는 캠퍼스로부터 도보로 걸리는 시간(분). 5, 10, 15, 20, 30 중 하나를 선택
  - [x] 설명, 참고 링크는 옵션
  - [x] 유효하지 않은 값 입력 시 텍스트 박스 하단에 에러 표시
  - [x] 새로고침해도 추가한 음식점 정보들이 유지해야한다.
- [x] 기타 UX
  - [x] dimmer 클릭시 모달 닫힘
  - [x] 모바일 반응형으로 제작한다.
